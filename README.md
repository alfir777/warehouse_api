# warehouse api

Тестовое задание

## Запуск проекта

1. Клонировать репозиторий или форк

```
git clone https://github.com/alfir777/warehouse_api.git
```

2. Выполнить переименование файла .env_template на .env и выставить свои параметры

```
cd warehouse_api/
cp .env_template .env
```

3. Запуск

```
python3 main.py
```
